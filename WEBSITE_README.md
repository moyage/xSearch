# XSearch 介绍网页

## 🌐 访问方式

### 1. 本地 + 局域网访问
```bash
./start-server.sh
```
访问：http://localhost:8080（本机）或 http://你的IP:8080（局域网）

### 2. 临时外网访问
```bash
./start-ngrok.sh
```
生成公网 HTTPS 链接，任何人可访问

### 3. 永久外网访问
查看 `DEPLOY.md` 了解 GitHub Pages、Vercel、Netlify 部署方式
