Setup

Quick Install

Deploy in 30 seconds

Recommended

Global Install

Available in any OpenClaw project after installation

git clone https://github.com/moyage/xSearch.git
cd xSearch
npm install
npm run build
npm link
Alternative

Local Import

Direct local path reference, suitable for development

// In your OpenClaw task
task(skill='xsearch', prompt='Your search query')
Code

Usage Examples

A few lines of code to enable intelligent search

Basic Search

Simplest usage

// Quick fact lookup
task(skill='xsearch',
  prompt='Python 3.12 new features')

Advanced Config

Customize search parameters

task(skill='xsearch',
  prompt='Compare React and Vue', {
  options: {
    complexity: 'high',
    maxTokens: 8000,
    outputFormat: 'table'
  }
})

Batch Search

Search multiple topics simultaneously

task(skill='xsearch',
  prompt='Research AI Agent trends', {
  options: {
    mode: 'research',
    depth: 'deep'
  }
})
