方法一：npm link

将 XSearch 链接到全局，可在任何 OpenClaw 项目中使用

git clone https://github.com/moyage/xSearch.git && cd xSearch
npm install
npm run build
npm link

优点：全局可用，更新方便

快速

方法二：直接引用

在 OpenClaw 代码中直接引用本地路径

import { xsearchSkill } from 
  '/Users/mlabs/Programs/xsearch/dist/skills/xsearchSkill';

// 使用
task(skill='xsearch', prompt='搜索任务')

优点：无需构建，即引即用

配置

方法三：配置导入

在 OpenClaw 配置文件中添加 Skill 路径

{
  "skills": [
    {
      "name": "xsearch",
      "path": "/path/to/your/xSearch"
    }
  ]
}

优点：配置化管理，团队协作

🚀 快速开始

// 基础搜索
task(skill='xsearch', prompt='搜索 Python 3.12 新特性')

// 高级配置
task(skill='xsearch', prompt='对比分析 React 和 Vue', {
  options: {
    complexity: 'high',
    maxTokens: 8000,
    outputFormat: 'table'
  }
})

// 批量搜索
task(skill='xsearch', prompt='批量搜索：["AI趋势", "Web3发展", "量子计算"]')

// 深度研究
task(skill='xsearch', prompt='深度研究 AI Agent 发展趋势', {
  options: {
    mode: 'research',
    depth: 'deep'
  }
})
